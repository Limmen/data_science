# pregel

## About 
Naive pregel implementation in Scala using Akka for the bulk-synchronous message passing model.

Example implementation of PageRank algorithm in the pregel model.
## How to run

```bash
$ sbt
$ compile
$ run --vertex 1 --workers 2
```
